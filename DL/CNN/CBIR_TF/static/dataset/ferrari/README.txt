README for the Ferrari test data set
====================================

Author:  Dr. Mathias Lux, mlux@itec.uni-klu.ac.at
Date:    2012-02-28
License: Attribution-ShareAlike License

The test data set contains 100 images of sports cars downloaded from 
Flickr. For each photo an XML file exists, where you can find all the 
metadata available on the web page. All photos are licensed either with 
the attribution creative commons license or a free to use license. You 
can find the license type as well as the attribution in the XML files.

license 7: No known copyright restrictions, 
           http://flickr.com/commons/usage/
		   
The images have been classified based on the color of the car depicted. 
Classification can be inferred from the sub directories names.
